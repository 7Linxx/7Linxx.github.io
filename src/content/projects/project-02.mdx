## Descripción del proyecto

SITP es un aplicativo de escritorio que simula y optimiza rutas dentro del Sistema Integrado de Transporte Público de Bogotá. Usando algoritmos de búsqueda de caminos más cortos, el sistema permite a los usuarios encontrar la ruta más eficiente entre dos puntos dentro de la red de transporte.

Este fue un proyecto de equipo desarrollado junto a Nataly Manrique, Gabriela Poveda, Santiago Muñoz, Felipe Acosta y Kevin Tunjuelo.

## Mi aporte

Me encargué del desarrollo del **frontend en Java**, construyendo las vistas e interfaces gráficas con las que el usuario interactúa para ingresar origen, destino y visualizar las rutas resultantes. También implementé el sistema de **properties para idiomas**, permitiendo que el aplicativo soporte múltiples idiomas de forma dinámica.

## Tecnologías utilizadas

El proyecto fue desarrollado completamente en **Java**, aprovechando sus librerías gráficas para la interfaz de usuario y estructuras de datos para la implementación de los algoritmos de rutas.
